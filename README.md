社区在2026.03.25发布了全新的社区调整方案，其做出的各种限制以及先前对EDU的限制已然让CodemaoEDUTools成为了一个废物工具。

于我先前预言的内容一致，CodemaoEDUTools一定不会活的长久，现在也算是有了结局，我也不想再弄下去了。

从23年发布的第一版，到25年的重置，再到今天的停更。不知道有多少人会看见，不知道有多少人在用。但只要是有人用了，有人知道，我就心满意足了。

编程猫EDU已经成为过去式了，亲手终结的一个时代，编程猫到底死过多少次了呢？倒也算是个未知数。

因为最近工作繁忙，我也不会再测试哪些功能能用哪些功能不能用了，2.20就是最后一个版本了，希望各位能Enjoy it。

七年编程猫，就此结束吧。
